# 麻雀スコア帖

4人麻雀の最終持ち点から順位点・累計ポイントを計算する、スマートフォン向けPWAです。

## 使い方

静的ファイルをHTTPサーバーで配信し、`index.html`を開いてください。外部ライブラリやビルド作業は不要です。

- 大会ごとに固定4人と計算ルールを登録
- 半荘の点数合計・100点単位・最終pt合計を検証
- 半荘の編集・削除、累計順位、ポイント推移グラフ
- LocalStorage保存、Service Workerによるオフライン対応

## 標準ルール

- 持ち点合計: 100,000点
- 30,000点返し
- 1,000点 = 1pt
- 順位点: +40 / +10 / -10 / -20
- 同点時: プレイヤー登録順を優先
